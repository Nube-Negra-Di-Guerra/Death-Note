# Death Note

## Unleash Your Inner Kira (For Fun Only)

This project is a simplistic, interactive web application that brings the concept of the Death Note from the popular manga series to a digital format. Designed with a high-contrast, gothic aesthetic, it allows users to "write" names and specify their demise, all within a browser-based interface. This application is purely for entertainment purposes and does not endorse any real-world harm.

## Features

*   **Write Names**: Add names to your digital Death Note.
*   **Specify Demise**: After adding a name, a modal dialog appears, allowing you to dictate the circumstances of their end (up to 500 characters). If no details are provided, the individual will meet their fate via a "Heart Attack in 40 seconds" by default.
*   **Erase Entries**: Easily remove names from the list.
*   **Persistent Storage**: All entries are saved locally in your browser, so your list remains intact even if you close and reopen the application.
*   **Thematic Design**: A high-contrast dark theme with custom gothic and handwritten fonts (UnifrakturMaguntia, IM Fell English, Kalam, Lora) to immerse you in the Death Note universe.
*   **Theme Toggle**: Switch between a dark "notebook" mode and a light "white paper" mode with a subtle moon/sun icon toggle at the bottom-right of the screen.
*   **Responsive Layout**: The application adapts to different screen sizes, ensuring a consistent experience on desktops, tablets, and mobile devices.
*   **Lore-Accurate Rules**: The rules displayed within the notebook are inspired by the original manga, adapted for a fun, digital context.

## The Rules of the Death Note

*   The human whose name is written in this note shall die.
*   If the cause of death is not specified, the person will die of a heart attack in 40 seconds.
*   Details of the death should be written in the next 6 minutes and 40 seconds.

## Technologies Used

This project is built using fundamental web technologies, ensuring simplicity and broad compatibility:

*   **HTML5**: For structuring the content.
*   **CSS3**: For styling and thematic presentation, including responsive design and custom fonts.
*   **JavaScript**: For all interactive functionalities, such as adding/removing entries, modal handling, and theme toggling.

No external frameworks, libraries, or complex integrations are used, keeping the codebase lean and easy to understand.

## Usage

1.  **Enter a Name**: Type the name of your target into the input field.
2.  **Write / Execute**: Click the "WRITE" button or press `Enter`.
3.  **Specify the End**: A modal will appear. You can either:
    *   Type a detailed description of their demise (up to 500 characters) and click "Seal Fate."
    *   Click "Heart Attack" to use the default 40-second heart attack.
4.  **Erase**: Click the "Erase" link next to any entry to remove it from the list.
5.  **Toggle Theme**: Click the moon/sun icon at the bottom-right to switch between dark and light modes.
